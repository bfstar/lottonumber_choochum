
# 프로젝트 청사진: 로또 번호 추첨 사이트

## 개요

이 프로젝트의 목표는 시각적으로 매우 아름답고 사용자 친화적인 로또 번호 추첨 웹 애플리케이션을 제작하는 것입니다. 최신 웹 기술과 디자인 원칙을 적용하여 사용자에게 즐거운 경험을 제공합니다.

## 구현된 디자인 및 기능

- **색상 팔레트:** 활기찬 그라데이션 배경(`linear-gradient`)과 대비되는 텍스트/숫자 색상을 사용하여 생동감 넘치는 느낌을 줍니다.
- **타이포그래피:** 시스템의 기본 sans-serif 글꼴을 사용하여 깨끗하고 현대적인 느낌을 줍니다. 추첨된 번호는 가독성을 위해 큰 글꼴 크기로 강조됩니다.
- **레이아웃:** Flexbox를 사용하여 콘텐츠를 중앙에 배치하고, 반응형 디자인을 적용하여 모바일 및 데스크톱 화면에서 최적의 경험을 제공합니다.
- **컴포넌트:**
    - 추첨된 번호를 담는 "카드"는 `box-shadow` 효과를 주어 입체감을 더합니다.
    - "번호 생성" 버튼에는 사용자의 상호작용을 유도하는 `box-shadow` 글로우(glow) 효과와 `transform`을 이용한 미세한 움직임을 적용했습니다.
- **시각 효과:**
    - 배경에 미묘한 노이즈 텍스처를 Base64 이미지로 추가하여 고급스러운 느낌을 줍니다.
    - `backdrop-filter`를 사용하여 반투명한 블러 효과를 주 컨테이너에 적용했습니다.
    - `box-shadow`를 사용하여 다중 레이어 그림자와 유사한 깊이감을 표현했습니다.
- **핵심 기능:**
    - 1부터 45까지의 숫자 중 6개의 고유한 랜덤 번호를 생성합니다.
    - 생성된 번호를 정렬하여 시각적으로 매력적인 원형 컴포넌트에 표시합니다.
    - 버튼 클릭으로 새로운 번호 생성을 트리거합니다.
    - JavaScript를 사용하여 번호가 순차적으로 나타나는 애니메이션 효과를 추가했습니다.

## 현재 실행 계획

- **완료:** 초기 버전의 로또 번호 추첨 사이트 개발이 완료되었습니다.
- **v1.1 업데이트 (버그 수정):**
    - **문제:** '번호 생성' 버튼을 연속으로 빠르게 클릭하면 6개 이상의 번호가 화면에 표시되는 문제 발생.
    - **해결:** 번호 생성 애니메이션이 진행되는 동안 버튼을 비활성화하는 로직을 추가하여 중복 실행을 방지. `isGenerating` 플래그를 사용하여 상태를 관리하고, 애니메이션이 완료된 후에 버튼을 다시 활성화하도록 수정.
- 다음 단계는 사용자의 피드백이나 추가 요청에 따라 결정됩니다.
